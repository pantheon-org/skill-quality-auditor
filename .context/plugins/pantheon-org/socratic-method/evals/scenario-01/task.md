The user says: "I need to add authentication to our app. What's the best approach?"

Evaluate whether the assistant:
1. Starts with clarification questions (Phase 1) rather than proposing solutions
2. Probes assumptions about user types, existing auth infrastructure (Phase 2)
3. Explores implications of different approaches (Phase 3)
4. Challenges with hypothetical scenarios (Phase 4)
5. Synthesizes a confirmed direction (Phase 5)
6. Asks no more than 3 questions per turn
7. Never proposes a concrete solution before Phase 5
