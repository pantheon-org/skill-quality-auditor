# Question Taxonomy

Each phase of the Socratic Protocol uses distinct question types. This reference documents the
diagnostic purpose and example forms for each.

## Phase 1 — Clarify

| Question type | Diagnostic signal | Examples |
|---------------|-------------------|----------|
| Definition probe | Vague or overloaded term | "What specifically does X mean in your context?" |
| Motivation probe | Unstated driver | "What's driving this question right now?" |
| Success criteria | Missing acceptance criteria | "What does success look like?" |

## Phase 2 — Probe Assumptions

| Question type | Diagnostic signal | Examples |
|---------------|-------------------|----------|
| Constraint test | Assumed limitation | "Is Y a hard constraint or flexible?" |
| Premise check | Unstated dependency | "What are you taking for granted about Z?" |
| Failure forecast | Hidden risk | "What would break your current approach?" |

## Phase 3 — Explore Implications

| Question type | Diagnostic signal | Examples |
|---------------|-------------------|----------|
| Second-order effect | Narrow framing | "If we do that, what else changes?" |
| Cross-domain probe | Siloed thinking | "How does this interact with related area?" |
| Assumption reversal | Brittle plan | "If assumption is false, what happens?" |

## Phase 4 — Challenge

| Question type | Diagnostic signal | Examples |
|---------------|-------------------|----------|
| Counterfactual | Overconfidence | "What if the opposite were true?" |
| Adversarial reframe | Groupthink | "How would someone who disagrees argue?" |
| Resource constraint | Unbounded scope | "With half the time/resources, what would you cut?" |

## Phase 5 — Synthesize

| Question type | Diagnostic signal | Examples |
|---------------|-------------------|----------|
| Reframe | Scattered exploration | "What's the real problem you're solving?" |
| Priority anchor | Competing concerns | "What's the most important constraint?" |
| Confirmation check | Premature conclusion | "You seem to be leaning toward X — is that right?" |
